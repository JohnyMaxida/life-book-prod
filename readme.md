## Описание

Life-Book - это Telegram-бот для 28-дневного марафона саморазвития с дневником осознанности. Бот помогает участникам проходить ежедневные задания, отслеживать прогресс и развивать осознанность.

**Текущая версия**: beta 2 

## Возможности

- 28-дневная программа саморазвития
- Дневник осознанности
- Отслеживание прогресса
- Умные напоминания
- Реферальная система
- Анализ и отчеты
- Ролевая система (Игрок, Участник, Специалист, Предприниматель)

## Архитектура - дописать на основе текущего проекта!!!

Проект использует модульную архитектуру с разделением на слои:

### Основные компоненты

- `lifebook.py` - Точка входа в приложение
- `db_manager.py` - Управление базой данных
- `db_schema.py` - Модели данных
- `config.json` - Конфигурация приложения

### Слой логики

- `logical_new.py` - Основной координатор логики
- `marathon_logic.py` - Логика марафона
- `ai_manager.py` - Интеграция с ИИ
- `referral_logic.py` - Реферальная система
- `report_manager.py` - Генерация отчетов
- `cron_manager.py` - Управление задачами по расписанию

### Пользовательский интерфейс

- `ui_blocks.py` - Блоки интерфейса
- `command_handlers.py` - Обработчики команд
- `media_handler.py` - Обработка медиафайлов

### Утилиты

- `utils.py` - Вспомогательные функции
- `const.py` - Константы
- `lifeBlock.py` - Текстовые блоки
